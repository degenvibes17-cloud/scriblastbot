# ScriblastBot ✍️

**Turn one article into 7 platform posts in seconds.**

[@ScriblastBot](https://t.me/ScriblastBot) is a Telegram bot that writes full articles and repurposes them for Twitter (3 angles), LinkedIn, Telegram, Facebook, Threads, and Instagram — optimized per platform with correct hashtag strategy.

## ✨ Features

- **Full article writing** — Send a topic, get a structured article (Introduction → Body → Conclusion)
- **7 platform repurposes** — One /done turns your article into platform-optimized posts
- **3 tweet angles** — Hook, Alternative, and Authority per article
- **SEO-optimized hashtags** — Different strategy per platform
- **Custom brand voice** — Set your tone, style, and audience (Custom tier)
- **Custom formats** — Ad copy, email newsletters, scripts, etc. (Custom tier)

## 🚀 Quick Start

1. Open [@ScriblastBot](https://t.me/ScriblastBot) on Telegram
2. Send any topic or title
3. Get your full article
4. Send `/done` for platform repurposes

## 🛠 Tech Stack

- **Runtime:** Python 3 (telegram-bot-api library)
- **LLM:** AI language model for article writing
- **Storage:** Persistent JSON for user credits
- **Hosting:** VPS server
